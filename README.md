# Technology Discussion Web Application

A user-friendly forum to enable technology enthusiasts to create, discuss, and share knowledge on technology-related topics.

## Features

- User registration and login
- Create, edit, and delete discussion threads
- Post comments on existing topics
- Category-based discussion for better organization
- Search and filter discussions by tags/categories
- Moderation features for content management
- Responsive interface using PHP, Hack, and CSS

## Tech Stack

- **Backend:** PHP (79.7%), Hack (16.2%)
- **Frontend:** CSS (4.1%)
- **Database:** MySQL

## Getting Started

1. Clone this repository:
    ```
    git clone https://github.com/SATYAMP023/Technology-Discussion-Web-Application.git
    ```
2. Set up a local web server (e.g., XAMPP, WAMP)
3. Import the MySQL database schema from the `/database` directory
4. Update database credentials in the configuration files
5. Launch the application in your browser

## Usage

- Register a new user account
- Browse or create new discussion threads
- Reply to threads with comments
- Filter discussions by category or search for keywords
